# 代码贡献

如果您愿意为本文档提供帮助，首先表示感谢，请您按照以下步骤操作，并遵守下面的代码规范

## 拉取代码

前往文档的[源码仓库](https://github.com/UkiyoLee/TypeScript-Tutorial)，并点击 `fork` 将项目加入您的仓库中，然后在本地拉取您仓库中的文档项目，以便您进行修改和测试

## 通过 npm 下载依赖

使用 IDE 或 VS Code 打开您拉取的项目，并在项目根目录打开终端，输入以下命令

```terminal
npm install
```
